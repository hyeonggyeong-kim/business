# tasks.md

> 프로젝트: 독거 어르신 가족 공동 케어 앱  
> 팀: 7팀 (팀장: 정승혁 / 팀원: 구민우, 김형경)  
> 플랫폼: Android (Kotlin) + Wear OS  
> 최종 업데이트: 작업 시작 시점 기준

---

## 1. 개발 목표 요약

스마트워치(삼성헬스 API)에서 수집된 독거 어르신의 건강 데이터를 가족 전원이 동시에 확인하고, 이상 감지 시 가족 채팅과 역할 분담이 자동으로 연결되는 안드로이드 앱의 MVP를 개발하여 시연한다.

**시연 가능 최소 시나리오:**
워치 데이터 이상 감지 → 가족 앱 알림 발송 → 채팅방 대화 → 어르신 앱에서 가족 확인 표시

## 9. 최종 점검 체크리스트

### 기능 완성도
- [ ] 더미 워치 데이터가 서버에 정상 전송된다
- [ ] 이상 감지 조건(활동 없음 2시간, 심박 이상)이 서버에서 판단된다
- [ ] 가족 전원의 앱에 FCM 알림이 발송된다
- [ ] 알림을 탭하면 해당 채팅 스레드로 이동한다
- [ ] 채팅방에서 메시지 전송·수신이 동작한다
- [ ] 어르신 앱에 "오늘 N명이 확인했어요"가 표시된다

### UI/UX
- [ ] 모든 버튼·텍스트가 노인 접근성 기준(글자 크기 18sp 이상)을 충족한다
- [ ] 어두운 환경에서도 대비가 충분한 색상을 사용한다
- [ ] 앱 로딩 후 주요 화면까지 3단계 이내로 도달 가능하다

### 안정성
- [ ] 시연 시나리오를 실기기에서 3회 연속 오류 없이 완주할 수 있다
- [ ] 네트워크 지연 시 로딩 인디케이터가 표시된다
- [ ] 더미 데이터 백업이 준비되어 있어 API 연동 실패 시에도 시연 가능하다

### 발표 준비
- [ ] 슬라이드가 문제 → 솔루션 → 데모 → 비즈니스 모델 순서로 완성되어 있다
- [ ] 각 파트별 발표 담당자가 지정되어 있다
- [ ] "왜 기존 앱과 다른가?" 질문에 대한 답변이 준비되어 있다
- [ ] "수익화는 어떻게?" 질문에 대한 B2G 모델 설명이 준비되어 있다
- [ ] 발표 당일 APK가 기기에 설치되어 있고 인터넷 연결이 확인되었다
